# Plant Landing Page

A modern and responsive plant store landing page built using React, Vite, and Tailwind CSS.  
The project features a premium glassmorphism UI, responsive layouts, and modern product showcase sections inspired by a Figma design.

---

## Tech Stack

- React.js
- Vite
- Tailwind CSS

---

## Features

- Responsive modern UI
- Glassmorphism design
- Product showcase cards
- Customer review section
- Fully responsive layout
- Optimized component structure

---

## Setup & Run

### 1. Clone the repository

git clone https://github.com/Ayman-khan-061/modern-plant-store

### 2. Install dependencies

npm install

### 3. Start development server

npm run dev

### 4. Build for production

npm run build
